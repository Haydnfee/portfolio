Process and Techniques

With the quick selection tool (due to the contrast between foreground and background), i cut the model from the background and then filled the gap via content aware fill. Now that i have the model and background seporate i can clone a few extra model layers. The first i streched out to the left via liquify which will be used to create the bubble effect. Using a custom brush, i painted on/erased masks to make the dispersing bubbles. The last copy was given a black color overlay, then i blurred it and lowered the opacity to create shadow.

Why

Fashion is a big and versitile industry, and i wanted to try and edit a photo that could be seen in a magazine that's not some generic photo retouching. It was also a great excuse to play with painting/erasing masks and creating custom brushes.
